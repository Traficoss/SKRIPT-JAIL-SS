# Jail Skript 
Sistema de Jail e ScreenShare para servidores Minecraft usando Skript. 
 
## Features 
- Jail normal 
- Silent Jail 
- Bloqueio de comandos 
- Bloqueio de movimento 
- Chat local na jail 
- Detecta logout durante ScreenShare 
 
## Comandos 
/jail <player> 
/silentjail <player> 
/unjail <player> 
 
## Requisitos 
- Skript 
- Spigot ou Paper
